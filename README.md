# House Prices Data Preprocessing Pipeline

## Project Overview

This project demonstrates the creation of a robust machine learning preprocessing pipeline using the House Prices dataset.

The workflow includes:

* Identifying numerical and categorical features
* Splitting data into training and testing sets
* Handling missing values
* Encoding categorical variables
* Scaling numerical features
* Combining transformations using ColumnTransformer
* Building an end-to-end Pipeline
* Preventing data leakage by fitting preprocessing steps only on training data
* Saving the final pipeline using Pickle
* Verifying successful pipeline serialization and loading

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Pickle

## Key Learning Outcomes

* Feature preprocessing
* ColumnTransformer
* Pipeline construction
* Data leakage prevention
* Model deployment preparation
